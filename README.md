# IT Help Desk Ticketing System

## Overview

The IT Help Desk Ticketing System is a Python and SQLite-based application designed to manage and track IT support requests. The system enables ticket creation, status updates, priority management, and reporting.

## Features

* Create Support Tickets
* View All Tickets
* Search Tickets
* Update Ticket Status
* Delete Tickets
* Status Reporting
* Priority Analysis

* GitHub
